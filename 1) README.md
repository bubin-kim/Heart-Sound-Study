# Heart-Sound-Classification-Study
Dataset: Heart Sound Database (via Kaggle)
본 프로젝트는 캐글에서 제공되는 심음(PCG) 데이터를 활용하였습니다. 
해당 데이터는 심장 박동의 음향적 특성을 포함하고 있으며, 정상(Normal)과 비정상(Abnormal) 분류를 위한 학습용 샘플로 구성되어 있습니다.
심장 박동 사이에서 발생하는 심잡음(Murmur)의 주파수 특성을 시각적으로 식별.

심장 박동음으로 이상질환을 탐지하는 것에 관심이 있어 훈련용 데이터셋을 이용하여 혼자 공부해본 내용으로 부족한 부분은 계속해서 보완할 예정입니다.
