# Heart-Sound-Classification-Study
Dataset: Heart Sound Database (via Kaggle)  
본 프로젝트는 캐글에서 제공되는 심음(PCG) 데이터를 활용하였습니다.   

심장 소리 데이터가 정상과 비정상 상태에서 어떤 차이를 가지는지 궁금하여, 이를 직접 시각화하여 확인해보고자 진행하였습니다.  

심장 소리는 저주파 영역에서 중요한 특징을 가져 샘플링 레이트를 2000Hz로 설정하여 분석 수행하였습니다.

#### 시각화를 한 이미지 분석
정상 심음은 저주파 영역에 에너지 집중되어있고, 패턴이 일정하고 반복적이라 큰 튀는 구간 없음.
비정상 심음에서는 특정 구간에서 강한 신호가 나타나고, 전체적으로 패턴이 불규칙한 모습을 보인다.
